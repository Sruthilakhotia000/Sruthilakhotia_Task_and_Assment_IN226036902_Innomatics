# 🤖 GenAI Prompt Templates using LangChain

## 📌 Project Overview
This project focuses on building a **Mini Prompt Engine** using LangChain that converts user inputs into structured and dynamic prompts.  

Instead of using hardcoded strings, we leverage **PromptTemplate and ChatPromptTemplate** to create reusable, scalable, and flexible prompt systems.

---

## 🎯 Objective
To design and implement a **dynamic prompt generation pipeline** that:

- Replaces hardcoded prompts with reusable templates  
- Handles multiple input variables  
- Supports different prompt styles  
- Validates user inputs  
- Generates structured outputs dynamically  

---

## 🚀 Features

- 🔁 Reusable Prompt Templates  
- 🔀 Multi-input Prompt Handling  
- 🎨 Prompt Variations (Teaching, Interview, Storytelling)  
- 💬 Chat-based Prompt System  
- ✅ Input Validation Layer  
- ⚡ Dynamic Prompt Generator  
- ♻️ Template Reusability Testing  

---

## 🛠️ Tech Stack

- 🐍 Python  
- 🔗 LangChain  
- 📓 Jupyter Notebook / Google Colab
