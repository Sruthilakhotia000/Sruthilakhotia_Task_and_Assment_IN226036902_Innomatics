🤖 BERT Fine-Tuning for Sentiment Analysis

📌 Overview

This project demonstrates the fine-tuning of a pre-trained BERT model for sentiment classification using real-world text data. It highlights how transformer-based models can be effectively adapted for NLP tasks such as sentiment analysis.

---

🎯 Objective

- Perform text preprocessing on raw dataset
- Tokenize text using BERT tokenizer
- Fine-tune a transformer-based model for classification
- Evaluate model performance using standard metrics

---

📂 Dataset

The dataset used in this project is sourced from Kaggle:

🔗 https://www.kaggle.com/datasets/jp797498e/twitter-entity-sentiment-analysis

This project specifically utilizes:

- Twitter Training Dataset
- Twitter Validation Dataset

The data contains text labeled into sentiment categories such as positive, negative, and neutral.

---

🛠️ Technologies Used

- Python 🐍
- PyTorch 🔥
- Hugging Face Transformers 🤗
- Scikit-learn 📊
- Pandas & NumPy

---

⚙️ Key Features

- Fine-tuning of pre-trained BERT ("bert-base-uncased")
- Custom dataset class for efficient data handling
- Tokenization with padding and truncation
- Evaluation using accuracy, precision, recall, and F1-score
- Visualization using confusion matrix

---

📊 Results & Insights

- Successfully applied transfer learning for sentiment classification
- Model effectively captures sentiment patterns from text
- Demonstrates practical usage of transformer-based architectures
